**TASK:** Extract kinetic parameters and degradation data from scientific figures.

**SOURCES:** 1) Figure Image (PRIMARY), 2) Caption Text (CRITICAL for context)

---

## 🚨 CRITICAL: FREE ENZYME ONLY (Intrinsic Kinetics)

**Extract ONLY free/soluble enzyme data for protein sequence modeling.**

### ✅ ACCEPT: Free Enzyme Figures
Caption indicators:
- "Free enzyme", "Purified enzyme", "Soluble enzyme", "Native enzyme"
- Plain enzyme names without modifiers: "CotA", "Laccase", "Thermolysin"
- No mention of support/carrier/material

### ❌ REJECT: Immobilized/Modified Enzyme Figures
DO NOT extract if figure caption/legend contains:

**Trigger words (if ANY appear, SKIP the figure):**
- Immobilization: immobilized, supported, bound, attached, cross-linked, conjugated
- Materials: nanocomplex, nanoparticle, nanofiber, /PLE, /graphene, /polymer, /chitosan
- Devices: biosensor, electrode, reactor, microreactor

**Examples:**
- ❌ "Fig 3: Kinetics of Thermolysin/PLE50 nanocomplex" → SKIP ENTIRE FIGURE
- ❌ "Panel B: Immobilized CotA on graphene" → SKIP THIS PANEL
- ❌ "His6-OPH/PLE50 performance" → SKIP (complex)
- ✅ "Fig 2: Kinetic parameters of purified CotA laccase" → EXTRACT

### 📊 Comparative Figures (Free vs. Immobilized)
If figure compares conditions:
- ONLY extract data points/bars labeled "Free enzyme"  
- Skip all "Immobilized", "Nanocomplex", or material-labeled data
- Add note: "Excluded immobilized enzyme data from this figure"

---

## CRITICAL EXTRACTION RULES

1. **Caption FIRST** - provides enzyme names, types, and experimental context
2. **Separate enzyme name and type**:
   - "CotA laccase" → enzyme_name="CotA", enzyme_type="laccase"
   - Common types: laccase, peroxidase, oxidase, reductase, dehydrogenase, hydrolase
3. **One reaction = one record** - multiple products go in products array
   - ✅ products: [{"name":"epi-AFQ₁"}, {"name":"AFQ₁"}]
   - ❌ Don't split into multiple records
4. **Global context applies to all panels** unless explicitly stated otherwise
5. **Separate value/unit**: "15.5 μM" → Km_value:15.5, Km_unit:"μM"
6. **🔬 Biochemical parameter recognition**:
   - If caption shows "Vmax/E0" or "Vmax per enzyme" → Extract as **kcat_value** (NOT Vmax_value)
   - If caption shows "Vmax/(E0×Km)" → Extract as **kcat_Km_value**
   - These represent intrinsic enzyme kinetics: kcat = Vmax/E0

---

## FIGURE TYPE STRATEGIES

| Figure Type | Extract | Ignore |
|-------------|---------|--------|
| Michaelis-Menten/Lineweaver-Burk | Labeled Km, Vmax text | Raw data points |
| Bar Charts/Time-Course | Y-axis values, degradation % | - |
| pH/Temp Optimization | Peak value → optimal_ph/temp | - |
| Pathway Diagrams | Enzyme→Products relationship | Quantitative fields (set null) |

## CONFIDENCE SCORING (1-5)

- **5**: Labeled kinetic params + complete enzyme info + conditions
- **4**: Has params OR clear pathway, good caption context
- **3**: Enzyme + substrate, some values estimated from graph
- **2**: Minimal data, vague caption, low-resolution
- **1**: Extremely unclear, cannot reliably extract

---

## WORKFLOW

1. **Read caption** - identify figure ID, enzyme name+type, organism, conditions
2. **Determine figure type** - use appropriate extraction strategy
3. **For each panel/reaction**:
   - Apply global context
   - Extract enzyme_name (separate from type)
   - Extract substrate, products (all in one array)
   - Extract quantitative values (separate value/unit)
4. **Output JSON only** (no markdown blocks)
