# Mycotoxin Enzyme Kinetics Extraction - PDF Direct Mode v1.0

## Task
You are analyzing scientific paper images (PDF pages). Extract ALL enzyme kinetics and degradation data related to mycotoxin research from both text AND tables visible in the images.

## IMPORTANT: Multi-Page Analysis
- You will receive multiple page images from a single PDF document
- Analyze ALL pages together to build complete understanding
- Cross-reference information across pages (e.g., enzyme mentioned on page 1, kinetics data on page 5)
- Tables often span multiple pages - combine data from continuation tables
- Methods section may define conditions used throughout the paper

## What to Extract

### Extraction Triggers (ANY one)
- Has Km OR Vmax OR kcat OR kcat/Km values (in text OR tables)
- Has enzyme + substrate + degradation efficiency/products
- Has enzyme + substrate + EC number/UniProt ID
- Tables containing kinetic parameters

### Key Data Sources to Scan:
1. **Abstract** - Often summarizes key findings
2. **Results tables** - Primary source for kinetic parameters
3. **Text paragraphs** - Contains context, enzyme names, organisms
4. **Figure captions** - May contain supplementary data
5. **Materials & Methods** - Organism sources, conditions

## Critical Extraction Rules

### 1. Table Reading
- Read ALL columns and rows carefully
- Match column headers with values correctly
- If table continues on next page, combine data
- Standard deviations (±) should be noted but not included in the main value

### 2. Unit Conversion
- Keep original units from the paper
- Note unit in the appropriate *_unit field
- Common units: μM, mM, nM for Km; μM/min, U/mg for Vmax; s⁻¹ for kcat; M⁻¹s⁻¹ for kcat/Km

### 3. Multi-Entry Handling
- Each enzyme-substrate combination = separate record
- Different conditions (pH, temperature variants) for same enzyme-substrate = separate records if kinetic values differ
- Mutant vs wild-type = separate records

### 4. Context Inference
- Link enzyme names to their full descriptions
- Expand abbreviated species names (B. subtilis → Bacillus subtilis)
- Distinguish source organism from expression host

### 5. Confidence Scoring
| Score | Criteria |
|-------|----------|
| 5 | Complete data: kinetics + organism + conditions + IDs |
| 4 | Has kinetics + organism OR enzyme_full_name |
| 3 | Has kinetics, partial metadata |
| 2 | Sparse data, some key fields missing |
| 1 | Minimal/uncertain data |

## Output Format
- Return ONLY valid JSON array
- No markdown code blocks
- No explanatory text before or after
- Empty array [] if no relevant data found
